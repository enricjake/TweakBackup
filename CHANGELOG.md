## [0.1.0] - 2026-03-15

### Added
- **Export & Import utility**: Create shareable configuration backups and restore them easily on other systems.
- **One-Click Presets**: Pre-configured profiles to change the behavior of Windows instantly:
  - 🎮 **Gaming Mode** - System optimizations for high frame rates and low latency.
  - 💻 **Developer Mode** - Optimizations for power-users and coders.
  - 🔒 **Privacy Max** - Aggressive telemetry and tracking prevention.
  - ⚡ **Performance** - Disables visual effects for a snappier experience.
  - 🔋 **Battery Saver** - Preserves battery life.
- **Manual Configuration Panel**: Visually configure settings like Dark Theme, Hidden Files, File Extensions, and more.
- **Backup & Restore**: Automatically create a System Restore point before applying changes.
- **Graphical User Interface (GUI)**: User-friendly Tkinter dashboard.
